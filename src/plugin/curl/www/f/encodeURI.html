encodeURI - URL-encode an URI, preserving its structure

encodeURI( str:string ) :string
This is part of the curl plugin. You must load the plugin via either -lcurl on the shell command line, or a plugin curl; line in a script.

The encodeURI() function converts spaces and non-ASCII characters to %XX sequences. It does not convert other characters such as "?" and "&". This function is most useful when you have a URL that uses (or may use) non-ASCII characters in the resource name.

Examples
encodeURI("http://maps.br/São Paolo/") "http://maps.br/S%C3%A3o%20Paolo/" The ã is converted to %C3%A2 because those two bytes represent the UTF8-encoding of ã. The space is converted to %20.
Notes
  • The encodeURIComponent() function is better when appending a parameter to a URL, because it encodes more characters such as & which can't be included in a parameter value without this encoding.
