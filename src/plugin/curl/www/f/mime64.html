mime64 - Convert a string to MIME base64 encoding

mime64( str:string ) :string
This is part of the curl plugin. You must load the plugin via either -lcurl on the shell command line, or a plugin curl; line in a script.

This function converts a string to MIME base64 encoding. This is most prominently used to encode the username and password for an HTTP "Authentication: Basic" header line.

Examples
mime64("This is a test") "VGhpcyBpcyBhIHRlc3Q=" Each group of 3 bytes (not characters) of the string are converted to a group of 4 base-64 digits. These digits are upper- and lower-case ASCII letters, digits, and the "." and "/" characters. If the string's length isn't an exact multiple of 3 then the base-64 representation will be padded with one or two "=" characters.
const headers = [ "Authentication: Basic " + mime64(userid + ":" + password) ] This shows how you can build an "Authentication: Basic" header line. It's wrapped in square brackets to make it an array because that's how the curlGet() and curlPost() functions want to receive their headers.
Notes
  • "Encoded" doesn't mean "encrypted"! The only reason to use base-64 encoding is to convert potentially troublesome characters or random bytes into nice safe printable ASCII characters.
