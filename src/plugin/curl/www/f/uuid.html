uuid - Generate a unique ID string

uuid() :string
This is part of the curl plugin. You must load the plugin via either -lcurl on the shell command line, or a plugin curl; line in a script.

The uuid() funnction generates a Universally Unique Identifier string, consisting of 32 hex digits with some hyphens inserted to divide the value into groups of 8-4-4-4-12 digits. Many standards use UUIDs to identify documents, records, transactions, etc.

The digits are all random. If possible, they'll be derived from /dev/random or /dev/urandom. If that isn't available, it'll use the lrand48(3) function which doesn't really have enough bits to really generate 32 hex digits (128 bits) randomly.

The full UUID standard specifies multiple ways to generate the UUID value (e.g. by deriving some bits from a timestamp or counter instead of random), but this uuid() function is always random.

Examples
uuid() "0556cafe-1443-45d5-67bc-32d403f41d52"
Notes
  • UUIDs are also sometimes refered to as GUIDs ("Globally" instead of "Universally") especially in the Microsoft world.
  • Most standards say that UUIDs should use lowercase a-f as hex digits, which this uuid() function does. They also say that comparisons should be case-insensitive though, like jx's = operator.
