arity - Return the number of arguments

arity(arg)
This is part of the example plugin. You must load the plugin via either -lexample on the shell command line, or a plugin example; line in a script.

This is just a trivial function, meant to show how functions can be implemented in a plugin. It returns a count of its arguments.

Examples
arity(1,true,"Howdy!") 3 Here, the arity() function was indeed called with 3 arguments.
