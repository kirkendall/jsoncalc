Example Plugin

The example plugin does nothing useful. It merely implements one function and one command, so you can see how that's done.

Options   Functions   Commands   Other Features

Options

Option Name Type Default Description
str string "Wow!" Example of a string option.
num number 4 Example of a number option.
bool boolean true Example of a boolean option.

Functions

Function Description
arity() Returns a count of the arguments passed into it.

Commands

Command Description
example Displays its argument in rainbow text. The argument can be literal text, or a parenthesized expression that returns the text to display.

Other Features

The example plugin doesn't add any other features.
