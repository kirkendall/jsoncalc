atan2 - Angle toward a given point

atan2(x:number,y:number):number
This is part of the math plugin. You must load the plugin via either -lmath on the shell command line, or a plugin math; line in a script.

Returns the angle toward a given point. This is similar to atan(x/y) except that atan2 takes the signs of x and y into account to select the correct quadrant.

Examples
atan2(1,-2) 2.67794504459 From the signs of the arguments, the function knows the angle must be in the 4th quadrant.
