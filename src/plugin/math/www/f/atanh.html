atanh - Inverse hyperbolic tangent

plugin math;
atanh( n:number ) :number
This is part of the math plugin. You must load the plugin via either -lmath on the shell command line, or a plugin math; line in a script.

Returns the inverse hyperbolic tangent of n.

Examples
atanh(1) 0.88137358702
