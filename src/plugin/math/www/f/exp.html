exp - Exponent (Math.E raised to the power n)

plugin math;
exp( n:number ) :number
This is part of the math plugin. You must load the plugin via either -lmath on the shell command line, or a plugin math; line in a script.

Returns the exponent (Math.E raised to the power n) of n.

Examples
exp(1) 2.71828182846
