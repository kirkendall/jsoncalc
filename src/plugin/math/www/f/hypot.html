hypot - Hypotenuse (square root of sum of squares)

plugin math;
hypot( n1:number, ... ) :number
This is part of the math plugin. You must load the plugin via either -lmath on the shell command line, or a plugin math; line in a script.

Returns the hypotenuse (square root of sum of squares).

Examples
hypot(3,4) 5 3*3 + 4*4 = 5*5
Notes
  • This isn't limited to 2 dimensions. You can pass as many arguments as you want.
