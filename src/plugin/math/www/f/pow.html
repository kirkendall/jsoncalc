pow - Raise base to the given power

pow(base:number,power:number):number
This is part of the math plugin. You must load the plugin via either -lmath on the shell command line, or a plugin math; line in a script.

Returns the result of raising base to the given power.

Examples
pow(2,10) 1024 pow(1.3,4.122) 2.94899813712
