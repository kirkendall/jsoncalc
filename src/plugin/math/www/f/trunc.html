trunc - Round toward 0

plugin math;
trunc( n:number ) :number
This is part of the math plugin. You must load the plugin via either -lmath on the shell command line, or a plugin math; line in a script.

Returns n rounded toward 0 — downward if positive, upward if negatrive.

Examples
trunc(1.4) 1
