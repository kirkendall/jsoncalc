Math Plugin

The math plugin defines most of the functions and constants expected in JavaScript's Math object.

Options   Functions   Commands   Other Features

Options

The math plugin doesn't have any options.

Functions

You can call the following functions like a "normal" function, or with a JavaScript-like "Math." prefix. For example, sqrt(2) and Math.sqrt(2) both do the exact same thing.

The abs(), random(), and sign() functions don't appear in the following list because they are implemented in the main JsonCalc library, not in this plugin.

Function Description
acos(n:number):number inverse cosine
acosh(n:number):number inverse hyperbolic cosine
asin(n:number):number inverse sine
asinh(n:number):number inverse hyperbolic sine
atan(n:number):number inverse tangent
atan2(x:number, y:number):number angle toward a given point
atanh(n:number):number inverse hyperbolic tangent
cbrt(n:number):number cube root
ceil(n:number):number ceiling (round up)
cos(n:number):number cosine
cosh(n:number):number hyperbolic cosine
exp(n:number):number exponent (Math.E raised to the power n)
floor(n:number):number floor (round down)
log(n:number):number natural logarithm
log10(n:number):number base-10 logarithm
log2(n:number):number base-2 logarithm
pow(base:number, power:number):number raise base to the given power
round(n:number):number round to nearest integer
sin(n:number):number sine
sinh(n:number):number hyperbolic sine
sqrt(n:number):number square root
tan(n:number):number tangent
trunc(n:number):number round toward 0
hypot(n1:number, ...):number hypotenuse (square root of sum of squares)

Commands

The math plugin doesn't add any commands.
