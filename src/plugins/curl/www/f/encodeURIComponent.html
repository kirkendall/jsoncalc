encodeURIComponent - URL-encode an parameter value

encodeURIComponent(string)
This is part of the curl plugin. You must load the plugin via either -lcurl on the shell command line, or a plugin curl; line in a script.

This function converts a string into a series of pure ASCII characters that are safe to use as the values of parameters appended to a URL.

Examples
encodeURIComponent("A Piña Colada & Thou") "A%20Pi%C3%B1a%20Colada%20%26%20Thou" The ñ is converted %C3%B1, exposing the two bytes that represent that character in UTF-8. The spaces are converted to %20. The ampersand (&) is converted to %26.
Notes
  • The encodeURIComponent() function converts more characters to %XX notation than encodeURI(), such as & which can't be included in a parameter value without this conversion. Use encodeURIComponent() when constructing a URI/URL.
