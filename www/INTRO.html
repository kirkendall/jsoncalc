What is JsonCalc?

Examples in a shell script Examples of interactive use How to download/install JsonCalc

JsonCalc is a C library for manipulating JSON data, and a program that uses that library. The program is intended to be used for interactively exploring JSON data, or used it in a shell script to generate JSON or convert it to other formats.

The program can be run interactively or as part of a shell script. It has a rather powerful set of features, using a mixture of JavaScript and SQL. There are also some unique added features but mostly it's intended to be close enough to JavaScript and SQL that if you're familiar with those two languages then you can be productive with JsonCalc right away.

JsonCalc began as an interactive data exploration tool, then it acquired functions and variables as ways to easily repeat common actions. From there, it grew just a bit more to become a useful scripting tool.

Even if you only use JsonCalc interactively, all of the scripting features are available. You might as well know about them.
