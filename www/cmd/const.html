const - Declare one or more constants

const key = value
Declare a constant. Constants are like variables, except that once declared, you can't assign them a new value.

For convenience, you can declare multiple constants with a single const command by using a comma to delimit them.

Using const in a global context (outside of a function body, basically) declares a global constant. const in a function body declares a constant that is only accessible within that function.

Examples
const pi=3.14159 pi 3.14159 pi = 3 Attempt to change const "pi" Declare a constant and then display it. If we try to assign it a new value, we get an error message.
const min=70, max=100 80 between min and max true Declare two constants on one line, and then use them.
Notes
  • Global constants are stored in global.consts, or consts within a function to access the local consts. In a global context, consts is a synonym for global.consts.
