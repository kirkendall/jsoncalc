default - Defines a default case within a switch statement

default:
This only works within a switch statement. It should be the switch's last option, and it labels the action to perform if none of the case labels matched.
Examples
switch ("S") {
case "Y": "yes"; break;
case "N": "no"; break;
default: "ambiguous";
} "ambiguous" The value "S" doesn't match case "Y" or case "N", so the default case is used and we get "ambiguous"; Note that since it is the last case, we don't need a break command at the end of it.
Notes
  • This command takes no arguments, and is terminated with a colon (":") instead of a semicolon (";").
