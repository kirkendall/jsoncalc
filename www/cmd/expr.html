expr - Output the result of an expression

expr
You don't need to use the print command to output a value. Simply use the expr as a command, and the result will be displayed.
Examples
current_time "19:24:17" The current_time symbol returns the current time as an ISO-8601 string. JsonCalc displays that string.
product(2..5) 120 Factorials again. Basically, any expression outputs its result, unless you use it as part of some other command in which case that other command decides the result's fate.
Notes
  • JsonCalc uses the = operator for both assignment and case-insensitive equality testing. This rarely causes confusion.
