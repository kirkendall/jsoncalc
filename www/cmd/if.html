if - Conditionally execute a command

if (condition) thencmd
if (condition) thencmd else elsecmd
The standard conditional statement, with optional "else" clause. The thencmd and elsecmd can be either a single command followed by a semicolon, or a group of commands enclosed in curly braces without a semicolon.
Examples
if (now % 2 == 0) "even"; else "odd" "even" The now variable is the current timestamp. The condition tests whether it is evenly divisible by 2, and then either runs either "even" or "odd", both of which output their string.
