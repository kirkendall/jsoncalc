print - Output data without formatting it

print value, ...
Given a comma-delimited list of expressions, this outputs each expression without any modification. In particular, strings are written as raw UTF-8 text, no padding is inserted between items, and no newline is added at the end. Data types other than strings are output pretty much the same as they would be without the print command.
Examples
print "\e[34mWow!\e[m\n" Wow! "\e" is an ASCII escape character, so "\e[34m" is an escape sequence for blue text. After "Wow!", another escape sequence turns off the color, and "\n" adds a newline since print won't add one by itself.
print "The current file uses ", sizeof(data), " bytes of RAM\n" The current file uses 3616 bytes of RAM An example showing a comma-delimited list of values in a single print command.
Notes
  • This command exists mostly as a convenient way to output strings without quotes or backslash escapes. You could achieve a similar effect via "-Ostring" but print is more convenient.
  • Although print doesn't automatically add a newline, the interactive "JsonCalc:" prompt does.
