return - Exit a function, possibly with a value

return
return expression
The return command exits a function. If you provide an expression then the result of that expression is used as the function's value; otherwise null is used.
Examples
return RETURNED A VALUE: null The return command isn't very useful outside of a function body.
function square(x) { return x * x; } square(7) 49 Here we define a function which returns a number multiplied by itself, and then we use that function.
