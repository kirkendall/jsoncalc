switch - Branch based on a computed value

switch (value) { ... }
The switch command evaluates an expression and compares the result to each case label within its code block. If a case matches, its commands are executed. If no case matches, and there's a default label in the code block, then those commands are executed.
Examples
switch (4) {
case 2:
case 3:
case 5:
case 7:
"prime";
break;
case 4:
case 9:
"square";
break;
default:
"boring";
} "square" The expression 4 matches case 4 so "square" gets executed.
