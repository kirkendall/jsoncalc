var - Declare one or more variables

var key
var key = value
Declare a variable, and optionally assign its initial value. If you don't assign it an initial value, then null is used.

For convenience, you can declare multiple variables with a single var command by using a comma to delimit them.

Using var in a global context (outside of a function body, basically) declares a global variable. var in a function body declares a variable that is only accessible within that function.

Examples
var i=17 i 17 i = i + 1 i 18 Declare a variable and then display it, change it, and display it again.
var vmin=70, vmax=100 80 between vmin and vmax true Declare two variables on one line, and then use them.
Notes
  • Global variables are stored in global.vars, or vars within a function to access the local vars. In a global context, vars is a synonym for global.vars.
    vars {"i":18,"vmin":70,"vmax":100} vars.keysValues()
    keyvalue
    i18
    vmin70
    vmax100
     First we dump vars. Since we're in a global context, this dumps global variables. Next we use keysValues() to convert the vars object into a table, which is displayed in a prettier format.
