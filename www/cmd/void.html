void - Evaluate an expression and discard the result

void expression
It is possible to write functions that have side-effects. If you want to run a function for its side-effect without displaying the returned value, then you can run it as a parameter to a void command.
Examples
var i = 0; function incr() i = i + 1; void incr() i 1 incr() null i 2 The incr() function increments i. The function will also return null but we don't care about that. When we invoke it with void its return value is discarded, but without void the null is displayed.
