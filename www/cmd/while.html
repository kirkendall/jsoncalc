while - Loop until a condition becomes false

while (condition) cmd
This is the standard "while" loop. It evaluates the condition, and if the condition is true then it runs cmd and loops to evaluates the condition again. The can be either a single command terminated with a semicolon, or a series of comments enclosed in curly braces without a final semicolon.
Examples
var x = 1; while ( x <= 4 ) { x; x = x + 1; } 1
