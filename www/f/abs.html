abs - Return the absolute value

abs( n:number ) :number
Math.abs( n:number ) :number
Returns the absolute value of a number. For the sake of compatibility with JavaScript, you may also pass the Math object as the first parameter though this function will ignore it.
Examples
abs(17) 17 abs(-17) 17 Positive numbers are unchanged. Negative numbers are made positiive.
Math.abs(-12.87) 12.87 You may pass Math as the first parameter, to make it look like JavaScript.
Notes
  • Most of the Math functions are missing. You can add them via the -lmath plugin.
