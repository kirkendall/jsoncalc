any - Test whether any items are true

any( test:boolean ) :boolean
This returns true if the bool argument is true for any row in a group, when used with the @ or @@ operator. If passed an array of booleans, it will return true if any element is true.

This is an aggregate function, meaning it accumulates the count over a series of table elements when used with the @ operator, or a SELECT clause. You may also pass it an array to have it accumulate results across the elements of that array.

Examples
any([false,false,true,false]) true One of the values in the argument array is true, so the result is true.
test.scores.groupBy("run") @ any(score < 20) ? run [1] This checks the test.scores table in the sample data, looking for runs where at least one row has a score less than 20. Only run 1 contains a score less than 20, so the return value is an array containing the value 1.
