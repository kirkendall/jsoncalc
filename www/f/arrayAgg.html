arrayAgg - Build an array one element at a time

arrayAgg( arg:any ) :array
This accumulates arg values in an array, and returns the array. If arg is null it will be skipped.

This is an aggregate function, meaning it accumulates its result over a series of table elements when used with the # operator, or a SELECT clause. You may also pass it an array to have it accumulate results across the elements of that array.

Examples
arrayAgg(1...5) [1,2,3,4,5] The 1..5 expression returns an array of integers from 1 to 5. When you pass an array to arrayAgg(), it adds each element to a new array so you get a copy of the argument array back.
arrayAgg([1,"a",true,null,[],{}]) [1,"a",true,[],{}] Any element value except null gets added to the result array.
Notes
  • Honestly, its hard to think of things that arrayAgg() can do that the # operator couldn't do better. I implemented this mostly because MySQL has a similar function.
