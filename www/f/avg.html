avg - Return the average (arithmetic mean)

avg( num:number ) :number
This returns the average value of its num argument.

This is an aggregate function, meaning it accumulates its result over a series of table elements when used with the # operator, or a SELECT clause. You may also pass it an array to have it accumulate results across the elements of that array.

Examples
avg([1,2,3,6]) 3 This returns the average value of the list of numbers.
test.scores.groupBy("run") # {run, avg(score)}
runavg(score)
120
230
331.66666666673
 Here we're taking the test.scores from the sample data, splitting it into groups based on the value of the "run" column, and building an object for each group containing the run number and average score.

Note that the # operator is smart enough to compute separate averages over each group ("run" in this example).
