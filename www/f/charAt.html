charAt - Return a 1-character substring

charAt( str:string, pos?:number ) :string
Return a 1-character substring from str. If no position is passed, then return the first character. If a character position num is passed, then skip that many characters into str and return the next character. If no such character exists, return null.
Examples
"Steve".charAt() "S" With no position number, it returns the first character.
"Steve".charAt(3) "v" With a position number, it counts characters to find the one to return. The first character is at position 0.
