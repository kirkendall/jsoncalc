distinct - Remove duplicates from an array

distinct( arr:array, strict?:boolean, columns?:string[] ) :array
Return a copy of array arr with duplicates removed.

Normally it only compares adjacent elements when looking for duplicates. If the optional strict argument is true then it looks for duplicates anywhere in the array.

Two elements are normally considered to be duplicates if the === operator says they're equal. When dealing with objects, === compares all members, and it compares strings case-sensitively. You can change that behavior by passing columns — a member name as a string, or a list of member names as an array of strings. If the values of the corresponding members are strings, then those strings will be compared case-insensitively. Members not listed in columns will not be tested.

Examples
[1,2,3,3,1,2,4].distinct() [1,2,3,1,2,4] The second 3 was deleted as a duplicate because it came immediately after the first 3. The second 1 and second 2 were not detected.
[1,2,3,3,1,2,4].distinct(true) [1,2,3,4] All duplicates were detected, since the strict argument is true.
Notes
  • In an SQL-like "SELECT" statement, the "DISTINCT" keyword is transformed into a call of this function.
  • This is not an aggregate function.
