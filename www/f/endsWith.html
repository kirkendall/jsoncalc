endsWith - Test whether a string ends with another

endsWith( str:string, substr:string, ignoreCase?:true ) :boolean
The endsWith function returns true if the first string ends with the second, or false otherwise.

The comparison is normally case-sensitive, but you can add a true argument to make it case-insensitive.

Examples
"The Universe".endsWith("verse") true Now we know how "The Universe" ends. It seems appropriate.
"The Universe".endsWith("whimper") false Nope. "whimper" does not appear at the end of "The Universe".
Notes
  • JavaScript's version of endsWith() is case-sensitive; it doesn't support a case-insensitive flag. Adding true to make it case-insensitive is strictly a JsonCalc thing.
