fromCharCode - Convert character codepoints (numbers) to a string

fromCharCode( arg:number|string|array, ... ) :string
Convert character codepoint numbers to a string.

This accepts a variable number of arguments. All are converted and integrated into the returned string. They may be:

  • Numbers. Each number is a Unicode codepoint, converted to a single character.
  • Strings. Strings are just concatenated into the result string.
  • Arrays. Each element may be a number or string, processed as above.

(JavaScript supports a variable number of numeric arguments, but not strings or arrays.)

Examples
fromCharCode(89, 101, 115) "Yes" Those are the ASCII/Unicode codepoints for the letters "Y", "e", and "s".
fromCharCode([89, 101, 115]) "Yes" Same thing, but passed as a single array.
fromCharCode([89, 101, 115], "terday") "Yesterday" Mixing arrays, numbers, and strings.
"SPARKLE!".charCodeAt(1...4).fromCharCode() "PARK" Here, charCodeAt() generates an array of character codepoints from the middle of a string, and fromCharCode() reassembles them to recreate the substring.
