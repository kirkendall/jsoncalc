getenv - Fetch the value of an environment variable

getenv(str)
Fetch the value of an enviroment variable. The str parameter is an expression providing the name of variable to fetch. The name must be a string, and the value will also always be a string unless no such environment variable exists, in which case null is returned.
Examples
getenv("TERM") "xterm" Fetch the value of the $TERM environment variable.
getenv($PAGER.toUpperCase()) "-s1" First fetch the value of the $PAGER environment variable, which indicates your preferred pager program — usually either "less" or "more". Both of those programs allow you to store default setting in the environment as uppercase versions of their names, so the above expression should output the default settings for your preferred pager — either $LESS or $MORE.
Notes
  • The value is always a string. You can use parseInt() to convert a string to an integer.
    parseInt(getenv("COLUMNS")) 80 Fetch the value of $COLUMNS, and convert it to an integer. Many environments set $COLUMNS to the character width of the terminal.
