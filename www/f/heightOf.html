heightOf - Calculate the height of a string

heightOf(str)
This returns the height of a string. Usually this is 1, but if the string contains newline characters then it could be higher. If the string ends with a newline character, then that final newline character is ignored.
Examples
"typical".heightOf() 1 This example shows that heightOf() isn't always the same as length. The \U1f600 emoji is 2 columns wide, nudging the string height up a bit.
"".heightOf() 1 Even an empty string has a height of 1.
"Do\nA deer\n".heightOf() 2 This string is two lines high.
Notes
  • Basically the height is 1 plus the number of newline characters, except that if it ends with a newline then that final newline doesn't count.
