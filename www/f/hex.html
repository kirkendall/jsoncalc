hex - Convert an integer or string to hexadecimal

hex(int)
hex(int, length)
hex(str)
hex() returns a string of hexadecimal digits for an integer or the bytes of a string.

When passed an integer int without a length, it returns the hexadecimal number with a "0x" prefix, using the fewest possible hex digits.

When passed an integer int and length, it returns the hexadecimal number without a "0x" prefix, using the given number of hex digits.

When passed a string str, it converts each byte of the string to a pair of hex digits. Since jsoncalc uses UTF-8 internally, some characters may require more than one byte.

Examples
hex(42) "0x2a"
hex(42, 4) "002a"
hex(".MP4") "2e4d5034"
hex("\U0001f4a3") "f09f92a3" This is UTF-8 for you. \U0001f4a3 is a Unicode codepoint for the "bomb" emoji — a raw 32-bit character code. Encoding this particular character in UTF-8 requires 4 bytes, so that's what we're seeing here.
