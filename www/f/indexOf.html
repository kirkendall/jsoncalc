indexOf - Return position of first match

indexOf(arr, elem)
indexOf(arr, elem, true)
indexOf(str, substr)
indexOf(str, substr, true)
The indexOf function searches through either an array or a string, and returns the position of the first match. If no match is found, it returns -1.

When searching through an array, the second argument is compared to each element of the array. The subscript of the first match is returned. The comparison is similar to the === operator — you can search for complex values such as objects or nested arrays, not just numbers or strings.

When searching through a string, it looks for the first instance where the second argument appears as a substring of the first string, and returns its character index within the string. If the second string doesn't appear anywhere with the first, then -1 is returned;

When comparing strings (either as an element of an array, or as a substring within a string), comparisons are normally case-sensitive. You can append a true argument to make it be case-insensitive.

Examples
"I have a cat on my lap".indexOf("cat") 9 There are 9 characters before the first instance of "cat".
"I have a cat on my lap".indexOf("dog") -1 There word dog doesn't appear anywhere in the sentence.
"Once Upon a Time".indexOf("up") -1 "Once Upon a Time".indexOf("up", true) 5 Searching for "up" case-sensitively found nothing, but a case-insensitive search found it after 5 characters.
[1,2,3,"four","Five",3,2,1].indexOf(3) 2 The first instance of 3 occurs in at subscript 2.
[1,2,3,"four","Five",3,2,1].indexOf("five") -1 [1,2,3,"four","Five",3,2,1].indexOf("five", true) 4 A case-sensitive search failed to find "five", but a case-insensitive search found it.
[{x:1,y:2},{x:3,y:4},{x:5,y:6}].indexOf({x:3,y:4}) 1 Here's an example of searching for a complex value. The object {x:3,y:4} appears at subscript 1.
Notes
  • JavaScript has no case-insensitive option. Passing true as the third parameter is strictly a JsonCalc thing.
  • JavaScript's version of indexOf() can't search arrays for complex values.
