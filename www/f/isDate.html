isDate - Test for Date String

isDate( arg:any ) :boolean
Return true is arg is an ISO-8601 date string, else return false. Date strings look like "YYYY-MM-DD".
Examples
isDate("test") false isDate("2025-01-01") true "test" is a string but not a date string. "2025-01-01" is a date string
