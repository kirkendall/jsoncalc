isDateTime - Test for DateTime

isDateTime( arg:any ) :boolean
Return true is arg is an ISO-8601 dateTime string, else return false. DateTime strings look like "YYYY-MM-DDThh:mm:ss", possibly with a time zone appended. Time zones can be "", "Z", or "+hh:mm".
Examples
isDateTime(current_date) false isDateTime(current_datetime) true current_date is a built-in variable that always returns the current date, and current_datetime always returns the current dateTime. current_date is not a dateTime.
