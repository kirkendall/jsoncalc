isNaN - Test for NaN

isNaN(arg)
Return true is arg is not a number, else return false.
Examples
isNaN(1/0) true Most functions and operators return null when they fail. If you're expecting a number and get null instead, this can let you react to that.
Notes
  • In JavaScript, NaN is a special symbol that's returned in cases where a number is expected, but no number can be returned. It's short for "Not a Number". NaN isn't a legal JSON value though, so we use null instead. JsonCalc's isNaN() function returns true for any non-Number.
