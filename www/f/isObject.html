isObject - Test for Object

isObject( arg:any ) :boolean
Return true is arg is an object, else return false.
Examples
isObject({a:4}) true {a:4} is indeed an object.
isObject([1,2,3]) false jx differs from JavaScript in that arrays are not considered to be objects.
