isObject - Test for Object

isObject(arg)
Return true is arg is an object, else return false.
Examples
isObject({a:4}) true {a:4} is indeed an object.
isObject([1,2,3]) false JsonCalc differs from JavaScript in that arrays are not considered to be objects.
