isPeriod - Test for Period

isPeriod( arg:any ) :boolean
Return true is arg is an ISO-8601 period string, else return false. Period strings look like "PnYnMnWnDTnHnMnS", where n is a number and their following letter indicates the time unit: Years, Months, Weeks, Days, Hours, Minutes, or Seconds. If a number is 0 then it and its following letter are omitted. The "T" is required before hours, minutes or seconds, but can be omitted if all of those are zero/omitted.
Examples
isPeriod("P3D") true "P3" is indeed a valid period string, representing a period of 3 days.
