isTime - Test for Time

isTime( arg:any ) :boolean
Return true is arg is an ISO-8601 time string, else return false. Time strings look like "hh:mm:ss" or "hh:mm", possibly with a time zone appended. Time zones can be "", "Z", or "+hh:mm".
Examples
isTime("3:30pm") false isTime("15:00:00") true "3:30pm" is not a time string because it has an AM/PM indicator, but ISO-8601 times are always a 24-hour clock. Also the hour should be padded to 2 digits. "15:00:00" is a valid time.
