join - Combine an array of strings into a single string

join(arr)
join(arr, delim)
The join() function concatenates an array of strings together to form a single long string.

The delim argument is optional. If you omit it, "," is used as the default.

Examples
join(["life", "the universe", "and everything"]) "life,the universe,and everything" This example uses the default delimiter, which is just a comma with no spaces.
