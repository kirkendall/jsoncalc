keys - Return the names of an object's members

keys(obj)
This returns the keys (names) of an object's members, as an array of strings.
Examples
{foo:1,bar:2}.keys() ["foo","bar"] A simple example. "foo" and "bar" are the member keys in that object.
for (i = test.rps.keys()) "test.rps." + i + "=" + test.rps[i] "test.rps.rock=ロック"
"test.rps.paper=紙"
"test.rps.scissors=はさみ" test.rps is part of the sample data; it is an object with members named "rock", "paper" and "scissors". This example loops over those member names, and outputs the name and value of each one on a separate line, using a for loop and the [] subscripting on an object.
