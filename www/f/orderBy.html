orderBy - Sort a table

orderBy(table, columns)
This sorts a table via a given list of column names.

The table is simply an array of objects. Each object is a row, and the members of each rows' object serve as columns.

The columns list is an array of strings, corresponding to the object member keys (column names). You may also precede any field name by a true flag to indicate that that field should be sorted in descending order, instead of the usual ascending order.

For the sake of convenience, if you just want to sort by a single column in ascending sequence, you can just pass its name as a string, instead of an array of strings.

Examples
orderBy(emojis, "codepoint")
emojicodepointnamegroup
💣U+1f4a3bombmisc
💩U+1f4a9pile of poomisc
😀U+1f600grinning faceface
😕U+1f615confused faceface
🤣U+1f923rolling on the floor laughingface
🥳U+1f973partying faceface
The output is the "emojis" table, sorted by ascending codepoint. Since we're only sorting by a single column, we can pass "codepoint" as a simple string instead of an array of strings.
emojis.orderBy([true, "group", "name"])
emojicodepointnamegroup
💣U+1f4a3bombmisc
💩U+1f4a9pile of poomisc
😕U+1f615confused faceface
😀U+1f600grinning faceface
🥳U+1f973partying faceface
🤣U+1f923rolling on the floor laughingface
 This time we're sorting by descending "group", and ascending "name". Also, just for the sake of variety, this time we used JavaScript's object method notation for the function call.
Notes
  • This only works on arrays of objects (i.e., tables) because it uses names of object members to guide the sort comparisons.
