parseInt - Convert a string to an integer number

parseInt( arg:string|number ) :number
This converts a string or floating point number to an integer number.
Examples
parseInt("17") 17 Convert the string "17" to the number 17.
  • This supports hexadecimal numbers with a "0x" prefix, octal numbers with a "0o" prefix, and binary numbers with a "0b" prefix. It also supports octal numbers with a "0" prefix, like C, though JavaScript has deprecated that.
