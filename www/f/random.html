random - Return a random number

random( num?:number ) :number
Math.random( num?:number ) :number
This returns a random number. Without a num argument, it returns a random floating point number between 0.0 and 1.0 (though never exactly 1.0). With num, it returns an integer from 0 to num-1.

For the sake of compatibility with JavaScript, you may also pass the Math object as the first parameter, which will be ignored.

Examples
Math.random() 0.0416310015946 Returns a random number between 0.0 and 1.0.
random(6) 4 Returns a random integer from 0 to 5.
Notes
  • JavaScript's Math.random() function doesn't take any arguments. If you want a random integer in JavaScript, you have to use Math.floor(Math.random() * num). jx's random() function was extended to support a num argument because that's a common use pattern but jx doesn't have a floor() function by itself; floor() is part of the -lmath plugin.
