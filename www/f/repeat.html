repeat - Concatenate multiple copies of the same string

repeat(str, count)
The repeat() function returns a single string, formed by joining count copies of str.
Examples
repeat("Bah", 3) "BahBahBah" This example just repeats "Bah" 3 times. Simple.
test.scores@{run,name,histogram:repeat("%",score)}
