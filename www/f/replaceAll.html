replaceAll - Replace all instances in a string

replaceAll( str:string, searchstr:string, replacestr:string, ignoreCase?:true ) :string
replaceAll( str:string, regex:regex, replacepattern:string ) :string
Return a version of string str in which every instance of substring searchstr has been replaced by replacestr. The search is case-sensitive, unless you add an extra true argument.

Alternatively, return a version of string str in which every match of regular expression regex is replaced by a string derived from replacepattern. Matching will be case-sensitive unless you include a i flag on regex. replacepattern is a string which may include the following symbols to access all or part of the matching text.

SymbolWhat it's replaced by
$& or $0Copy of the matching text in str
$1 to $9Text matching 1st to 9th \(\)-wrapped portion of regex
$$A single dollar sign
Examples
"subject string".replaceAll("t", "T") "subjecT sTring" Every lowercase "t" is replaced by uppercase "T".
"subject string".replaceAll(/[aeiou]/, "($&)") "s(u)bj(e)ct str(i)ng" Every vowel is replaced by the same vowel wrapped in parentheses. You could do the same using replace() by adding a g flag to the regular expression, but replaceAll() doesn't need g.
Notes
  • This uses POSIX regular expressions, which are somewhat different from JavaScript regular expressions.
  • JavaScript doesn't support the final true parameter to make string searches be case-insensitive.
