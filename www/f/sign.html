sign - Return the sign of a number

sign( arg:number ) :number
Math.sign( arg:number ) :number
This returns the sign of its argument — 1 if positive, 0 if 0, -1 if negative.

For the sake of JavaScript compatibility, you may optionally pass the Math object as its first parameter. This is ignored.

Examples
Math.sign(-0.338432) -1 This returns -1 because the argument is negative.
Notes
  • Most of the Math functions are missing. You can add them via the -lmath plugin.
