sleep - Pause execution for a given duration

sleep( seconds:number|period ) :null
This pauses execution for a given number of seconds, and then resumes. It always returns null.

Usually you'll pass the sleep duration as a number, representing seconds. It can be floating point number, allowing sleeping by fractions of a second. Alternatively, you can pass an ISO 8601 period string such as "PT1M30S" to sleep for one minute and 30 seconds.

Examples
current_time;sleep(5);current_time "12:22:36"
null
"12:22:41" This first shows the current time, then sleeps for 5 seconds (showing the null returned by sleep(5)), and finally shows the current time again. You can see that 5 seconds elapsed between the two time checks.
current_time;sleep(0.5);current_time "12:23:18"
null
"12:23:18" This is like the first example, but we only slept for half a second so the current time didn't change significantly.
current_time;void sleep("PT1M");current_time "12:24:37"
"12:25:37" Here we're specifying the sleep time via an ISO 8601 period string. Also, we used the void command to hide the null returned by sleep().
Notes
  • At the time of this writing, JsonCalc's support for fractions of seconds in ISO time periods is limited. I really should do something about that. For now, though, if you need to sleep a fraction of a second then you must specify the sleep time as a number.
