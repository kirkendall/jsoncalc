slice - Return part of an array or string

slice(arr, start)
slice(arr, start, end)
slice(str, start)
slice(str, start, end)
The slice() function returns a new array filled with a portion of the given arr array, or a substring from string str.

If the start argument is 0 or more, then it indicates how many elements to skip from the start of arr, or characters from str. If start is negative, then it counts from the end of arr or str.

If end is omitted, then the slice extends to the end of arr or str. Otherwise, if 0 or positive then it indicates where the slice ends in arr or str; i.e., it is the index of the first element of arr to omit from the slice, or the first character of str to omit. If it is negative, then it counts from the end of arr or str instead of the beginning.

Examples
(0..10).slice(3,5) [3,4] The (0..10) returns an array of integers from 0 to 10. The .slice(3,5) skips the first 3 elements, and returns the text elements up to but not including the 5th element.
(0..10).slice(-1) [10] This time we start copying from 1 element from the end of arr. There is no end parameter so the slice extends to the end of arr.
"Billiards".slice(3, 7) "liar" Just a quick example to demonstrate that it works on strings.
