startsWith - Test whether a string starts with another

startsWith(str, substr)
startsWith(str, substr, true)
The startsWith function returns true if the first string starts with the second, or false otherwise.

The comparison is normally case-sensitive, but you can add a true argument to make it case-insensitive.

Examples
"The Universe".startsWith("big bang") false The string "The Universe" does not start with the string "big bang".
"The Universe".startsWith("The") true Yes, we have proven "The Universe" starts with "The"!
"The Universe".startsWith("the") false "The Universe".startsWith("the", true) true A quick example showing how case-insensitive tests work.
Notes
  • JavaScript's version of startsWith() is case-sensitive; it doesn't support a case-insensitive flag. Adding true to make it case-insensitive is strictly a JsonCalc thing.
