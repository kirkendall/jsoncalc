stringify - Convert data to a JSON string

stringify( data:any ) :string
JSON.stringify( data:any ) :string
This converts any data to a JSON string.
Examples
JSON.stringify(1..5) "[1,2,3,4,5]" The array [1,2,3,4,5] is converted to a string.
stringify({foo:1,north:true}) "{\"foo\":1,\"north\":true}" Here we're converting an object to a string. Since JSON requires quotes around member names, stringify() adds them. The backslashes are just how quotes are allowed in a quoted string. Also, note that we skip the "JSON." — JsonCalc allows it but doesn't require it.
Notes
  • This is sensitive to the "digits" and "ascii" attributes of the default formatting settings. You can adjust those settings via the -sformat command-line flag or set command. Other attributes of the default format are ignored, including "pretty".
  • The main difference between stringify() and toString() is that toString() leaves strings unchanged, while stringify adds quotes and backslashes as needed to make the value be valid JSON text.
