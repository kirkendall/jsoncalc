substr - Extract part of a string

substr(str, start, length)
substr(str, start)
This returns part of string str.

The start argument should be an integer. If it is 0 then the substring will start at the start of str If it is positive, then it indicates how many characters to skip from the start of str; if it is larger than the length of str then an empty string is returned. If start is negative, then it counts from the end of str; if it is larger than the length of str then the substring will start at the start of str.

The length is optional. If omitted, the substring will extend to the end of str. If zero or positive, it is the length of the substring; if that would extend past the end of str then it is ignored and the substring extends to the end of str again. Negative lengths are not supported.

Examples
substr("portion", 1, 3) "ort"
"portion".substr(-2) "on" This uses JavaScript's "arg.function()" notation. The start position is -2, and no length is specified, so it returns the last 2 characters of "portion".
