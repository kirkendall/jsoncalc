sum - Return the sum of a series of numbers

sum( num:number ) :number
This adds up all of the num argument values, and returns the sum.

This is an aggregate function, meaning it accumulates its result over a series of table elements when used with the # operator, or a SELECT clause. You may also pass it an array to have it accumulate results across the elements of that array.

Examples
sum([]) 0 An empty list adds up to 0.
sum(1...10) 55 This adds up all of the integers from 1 to 10. The "1...10" notation returns an array of integers connecting the two endpoints, so we're applying sum() to an array again.
test.scores.groupBy("run","overall") # {run, sum(score)}
runsum(score)
160
290
395
overall245
 This uses the test.scores table from the sample data, divides it into groups by the "run" column, and generates a row for each group with the run number and total scores for that run. The groupBy() function also added an extra row where the overall total could be computed.
