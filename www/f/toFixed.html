toFixed - Show a given number of digits after the decimal point

toFixed(num,digits)
This converts a number to a string with a given number of digits after the decimal point. Even if those digits are "0", they'll always be there.
Examples
toFixed(12.3, 2) "12.30"
12.3.toFixed(2) "12.30" Dots and decimal points can get a bit confusing when munged together like this.
