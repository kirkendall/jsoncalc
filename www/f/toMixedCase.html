toMixedCase - Capitalize words in a string

toMixedCase( str:string, exceptions?:array ) :string
This returns a copy of string str in which the first letter of each word is uppercase, and the remaining letters are lowercase.

You can also optionally pass an array of exceptions. Each element of the array can be one of two things:

  • A string containing a single word, capitalized the preferred way.
  • The symbol true to indicate that the first word is never an exception; this is handy for titles.
Examples
toMixedCase("bob DYLAN") "Bob Dylan" This doesn't care whether the input is uppercase, lowercase, or a mixture.
"PAUL MCCARTNEY".toMixedCase() "Paul Mccartney" toMixedCase() isn't great for "Mc" names.
"THE END OF THE WORLD".toMixedCase([true,"the","of"]) "The End of the World" Here's an example that uses an exception list. It forces "the" and "of" to be lowercase, except (because of the true) when they're the first word of str.
