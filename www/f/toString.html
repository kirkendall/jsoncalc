toString - Convert any value to a string

toString( arg:any ) :string
Returns a string version of arg.

If arg is already a string, it is returned unchanged. In particular, quotes will not be added, and no backslash escapes will be used.

Conversion of anything else is converted as JSON text, similar to the stringify() function. For numbers, booleans, and null this is trivial, but it can also handle objects and arrays.

Examples
toString("2024-10-31") "2024-10-31" Dates are strings, so they're returned unchanged.
toString(null) "null" Numbers and symbols are easy to convert.
toString({foo:2+2}) "{\"foo\":4}" Arrays and objects are converted to JSON. The backslashes shown in the response are not part of the string, that's just how jsoncalc displays strings. The same is true for the outermost set of quotes.
