widthOf - Calculate the width of a string

widthOf(str)
This returns the width of a string. This is different from the length because some Unicode characters are double-width, some are zero-width.
Examples
"Happy=\U1f600".widthOf() 8 "Happy=\U1f600".length 7 This example shows that widthOf() isn't always the same as length. The \U1f600 emoji is 2 columns wide, nudging the string width up a bit.
