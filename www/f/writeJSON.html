writeJSON - Write data out to a JSON file

writeJSON(data:any, file:string):boolean
This writes data out to the named file as JSON text. It returns true if successful.
Examples
void writeJSON({a:1,b:2}, "file.json") true This overwrites (if it exists) "file.json" or creates it (if it doesn't exist) and writes the data out as JSON.
Notes
  • I very deliberately avoided calling this "write()" because I may want to implement a POSIX plugin some day.
