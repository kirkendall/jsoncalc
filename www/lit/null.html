Null

"Null" is a weird concept. It is a value that signifies the absence of a value. In JsonCalc (and JSON and JavaScript), it is represented by the null symbol.

JsonCalc also uses the null data type to represent errors in expressions. Where JavaScript might use undefined or NaN, JsonCalc is limited to JSON data types, so null is all we get.
