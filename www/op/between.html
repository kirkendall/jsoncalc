BETWEEN operator

number
string		BETWEENlower AND upperTest whether number or string is between lower and upper
This is a ternary (three-operand) operator inherited from SQL. It is effectively the same as a pair of comparisons. It can work on numbers or strings. When comparing strings, the comparison will be case-insensitive since that's SQL convention.
Examples
2 between 4 and 10 false 7 between 4 and 10 true No surprises. Note that the "and" here is part of the "between" operator, and is not a logical-and.
SELECT * FROM test.scores WHERE score BETWEEN 30 AND 40

Notes
  • If the expression being tested is complicated, then BETWEEN might be more efficient than a pair of <=/>= comparisons, because BETWEEN only needs to compute the value once.
