Coalesce operator

expr??exprIf left expr is non-null, return it; else return right
This returns the first non-null operand. Often you'll use a whole series of ?? operators with values, as a prioritized list of possibilities to return.
Examples
test.contact.cell ?? test.contact.email ?? test.contact.home "mycontact@sample.com" In the sample data, test.contact has no "cell" member, but it does have "email" so we get that. It also has a "home" member, but since we found "email" we don't need "home".
Notes
  • This uses "short circuit" evalulation, meaning the right operand isn't evaluated if the left operand is non-null.
  • JavaScript has this too!
  • JsonCalc also has an =?? assignment operator which only assigns if the new value wouldn't be null, and a :?? notation in object generators to only add members if the value would not be null. JavaScript does not have these.
