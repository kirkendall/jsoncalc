These are comparison operators. They work on numbers or strings.

If you compare a number to a string, the string will be converted to a number and then the numbers will be compared. If the string can't be converted to a number then the comparison is false.

For strings, the comparison is case-sensitive. If you want to compare two strings case-insensitively, use toUpperCase() or toLowerCase() to convert them before comparison.

Examples
1 < 2 true Yes, 1 really is less than 2.
"Steven" >= "stephen" false "Steven".toUpperCase() >= "stephen".toUpperCase() true The codepoint for uppercase "S" is less than the codepoint for lowercase "s", so "Steven" comes before "stephen". However, by converting both strings to be entirely uppercase, the case difference goes away and "V" comes after "P".
