IS NULL operators

exprIS NULLTrue if expr is null
exprIS NOT NULLFalse if expr is null
These compare an expression to null.
Examples
glorxpik is null true If there is no variable named glorxpik, then it'll be treated as a null value, and the is null operator can detect that.
1/0 is null true Division by zero returns null.
Notes
  • These operators exist mostly for SQL compatibility. Honestly, I'd rather just use === null or !== null.
  • Its worth noting once again that since standard JavaScript symbols such as undefined and NaN aren't valid in JSON, JsonCalc uses null to represent exceptional conditions and errors.
